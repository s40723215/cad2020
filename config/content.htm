救救菜英文

在上程式課時常常需要用到英文，記得從一年級開始，老師就一直強調英文到底有多種藥，像前陣子我的網頁出了問題去github上找解決辦法但是超長的英文真的很容易讓人放棄。

我們學校的英文多益門檻是390分，老實說這根本不夠用，但最近大三同學們做了學校的第一次多益測驗，聽說考超過390的人幾乎都在10個以內，我的英文也沒有到很好，多益差不多是綠色證書的門檻，不過我想來分享我看到的學習英文不錯的辦法。

聽英文歌

 我覺得現在很多同學英文不好就是覺得看不懂、背單字又痛苦，因此我覺得先想辦法有學英文的慾望比較好，且我發現很多英文不好的同學也很愛聽英文歌

背單字

 背單字雖然枯燥，但是沒有一定的單字量，其他的方法用再多都是徒勞，雖然教育部有規定國中2000單、高中7000、高職4500單，但是很多同學連這些量都沒有達標。

看英文新聞或英文小說

 在有一定的單字量後，這個方法可以讓我們更快學習新單字，在閱讀完一篇文章後不認識的單字可以用前後文去猜意思，然後如果每天都讀一點的話，一些原本不知道的單字都會重複出現，久而久之就會記起來了。

聽新聞Podcast

就算多益很高分，英文還是拿來溝通的語言，因此空閒時間聽一些Times或cnn news我覺得對聽、說蠻有幫助，雖然很常聽不懂不過可以盡量聽。

Develop

https://github.com/mdecourse/cmsimde 的開發, 可以在一個目錄中放入 cmsimde, 然後將 up_dir 中的內容放到與 cmsimde 目錄同位階的地方, 使用 command 進入 cmsimde 目錄, 執行 python wsgi.py, 就可以啟動, 以瀏覽器 https://localhost:9443 就可以連接, 以 admin 作為管理者密碼, 就可以登入維護內容.

cmsimde 的開發採用 Leo Editor, 開啟 cmsimde 目錄中的 cmsimde.leo 就可以進行程式修改, 結束後, 若要保留網際內容, 只要將 cmsimde 外部的內容倒回 up_dir 目錄中即可後續對 cmsimde 遠端倉儲進行改版.

init.py 位於  up_dir 目錄, 可以設定 site_title 與 uwsgi 等變數.
